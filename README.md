# AI Cover Letter Generator

## Project Overview

This project is an AI-powered Cover Letter Generator built using React, Tailwind CSS, Gemini API, and PDF parsing.

The user can:

- Enter personal and job details
- Upload a Resume PDF
- Extract resume text automatically
- Generate an AI-based personalized cover letter
- Copy the generated letter instantly

This project was created as part of Week 4 – AI Engineering Assignment.

---

## Features

- Responsive modern UI
- Form validation
- Gemini AI integration
- Secure API key using `.env`
- PDF Resume Parsing using `pdfjs-dist`
- Loading animations
- Copy to Clipboard feature
- Personalized cover letter generation

---

## Tech Stack

- React + Vite
- Tailwind CSS
- Gemini API
- pdfjs-dist
